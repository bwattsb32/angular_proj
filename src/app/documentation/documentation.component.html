Documentation Menu

The Documentation menu allows the user to view document templates, generate reports from document templates, and download and save the generated reports. It displays the following sub-menus*.
  1. Assessments - This menu item takes the user to list and maintain all assessments. It allows the tracking of an assessment process and save all relevant information and statuses.
  2. Templates - This menu item takes the user to list and view all document templates. It also allows the user to generate reports, download and save them in the document repository.
  3. Documents - This menu item takes the user to list and view all reports in the document repository. It also allows uploading miscellaneous documents to the document repository.
* Not all menu items may be available to every user.
